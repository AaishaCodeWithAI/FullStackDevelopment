      Tables in HTML

      Tables in HTML are used to represents real-life table data. It consists
      rows and columns.

      • Examples of Table:
        • Student List
        Student ID Students Name Age Phone no Core Subjects
        LC110001 Roy Lame 21 9812983012 Chemistry Physic Math Biology
        LC110002 Sam Nale 23 9843920211 Chemistry Physic Math Computer
        LC110003 Roniya Sant 22 9846381161 Chemistry Physic Social Biology

      • Semantics in Tables

        Even the tables got semantic markup. It contains:

        • thead - to wrap table header.
        • tbody - to wrap table body.
        • tfoot - to wrap table footer

      • Table ColSpan & RowSpan Attributes

        Used to create cells which spans over multiple rows and columns.

        Forms are used to collect data from user.

          • Select your gender

        • Action Attribute

          It is used to define what action to be performed when the form
          is subitted or where the form data should be sent.

          For Example: <form action = "/action.php">

        • Form Elements

          • Input Elements

            Used to create multiple input controls, and there are
            multiple types of inputs that can be created using type attrtibute

              # Common Input Elements are:
            • Text
            • Number
            • Password
            • Date
            • Time
            • Color(rarely Used)
            • button

          • Placholder

            It is used to display a short hint or example text inside an input
            field or textarea before the user enters a value.It helps guide the
            user about the expected input format or type of data.

          • Label

            It represents the caption for an item in the user interface.

          • Button

            It is used to create a clickable button on a webpage. It can be
            used to submit forms, trigger JavaScript functions, or perform
            other interactive actions.

            We can also make a button through input element. But, It is not
            generally advisable, due to limited functionality, compatiblity
            issues, and more.

              # Type Attribute of Buttons are:
            • Submit
            • Button
            • Reset

          • Name Attribute

            Name of the form control. Submitted with the form as a part of
            name/value pairs.

          • Checkbox

            It is an input element that allows the user to select one or more
            options from a set of choices. It is created using the <input> tag
            with the attribute type="checkbox", and if we write checked in
            our input then by default the checkbox will come checked in
            hand.

          • Radio button

            an input element that allows the user to select only one option
            from a predefined set of choices.

          • Dropdown using Select Element

            It is created using the <select> element with nested <option>
            elements to provide a list of selectable choices. It's commonly
            used in formsto let users pick one or more values.

          • Range

            It is an input control that lets the user select a numeric value
            from a continuous range using a slider. It's often used when
            the exact value is not critical, but a value within a certain ra-
            -nge is needed — for example, adjusting volume, brightness,
            or zoom level.

          • Textarea

            It is a multi-line text input control, commonly used in forms to
            collect user input such as comments, feedback, or reviews. Un-
            -like single-line input fields, <textarea> allows users to input
            larger amounts of free-form text.
