  2. HTML Elements created using Tag

    Tags are the components used to design the structure of the webpages.They are special keywords enclosed in
    angle brackets (< >) that define the structure and meaning of content on a web page.

    Structure of an HTML Tag

    • Opening tag: () — starts an element.
    • Closing tag: ()— ends an element.
    • Self-closing tag: () — does not need a closing tag.

      Different Tags

    • Paragraph Tag

      It is the tag used to respresent paragraph. Using this tag, we can write pargaraph as this one that
      I wrote to showcase this line of text.

      We use ("<+p+><+/p+>") to write any paragraph.

    • Bold Tag

      It is the tag used to represent bolding. Using this tag, we can Bold any certain text for highlighting.

      We use ("<+b+><+/b+>") to bold any text.

    • Italic Tag

      It is the tag used to reprsent italic. Using this tag, we can slaht the text little for hightlighting.

      We use ("<+i+><+/i+>") to slaht any text.

    • Underline Tag

      It is the tag used to represent underline. Using this tag, we can underline the text little for highlighting.

      We use ("<+u+><+/u+>") to underline the text.

    • Heading Tag

      It is the tag used to represent Heading. Using this tag, we can write

      heading
      in diferent level for making our program
      more readble and visually pleasing.

      We use ("<+h1+><+/h1+>") to write heading. It is from h1 to h6 level. but only h1 to h4 and sometimes h5 is used.

    • Ordered List Tag

      It is the tag used to represent list which are ordered. Using this tag, we can order our list, in different type like number,
      uppercase and lowercase alphabates, roman numbers.

      We use ("<+ol+><+/ol+>") to write ordered list. And we use a tag called listing tag inside the ol for writing the points.

    • Unordered List Tag

      It is the tag used to represent list which are unordered. Using this tag, we can order our list in bullet points, to represent
      the points.

      We use ("<+ul+><+/ul+>") to write unordered list. And we use a tag called listing tag inside the ul for writing the points.

    • Anchor Tag

      It is the tag used to represent links in our pages. Using this tag, we can link any absoulte(Exact from Internet) or
      relative(Our other file links).

      We use ("<+a+><+/a+>") to link different pages to our pages.

      Relative File
      Absoulte Files
    • Image Tag

      It is the tag used to represent images in our pages. Using this tag, we can link images either absoulte or relative
      images.

      We use ("<+img+>") to link images to our pages. Image tag is a self closing tag. So, we don't need to close it.

    • BreakLine Tag

      It is the tag used to represent breaking the line in long paragraphs. It helps us to make our website more readable
      and visually pleasing.

      We use ("<+br+>") to break the line in our paragraphs. BreakLine Tag is a self closing tag too; same as the image tag.

    • HorizontalLine Break Tag

      It is the tag used to represent a thrematic break between paragraphs. It helps us to seperate section in efficient way.

      We use ("<+hr+>") for horizontal line break in our paragraphs. HorizontalLine Break Tag is also a self closing tag.
