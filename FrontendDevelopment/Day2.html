  1. Inline vs Block Elements

    • Block Elements

      Block Elements takes up the entire width no matter
      necessary or not for them, and they always begin from next line

    • Inline Elements

      Inline Elements only takes the width necessary for them,
      and they don't begin from next line

      Anchor Tags are Inline Element.

      w3school TutorialsPoint

      Image tags are also Inline Element.

  2. Div vs Span Element

    • Div Element

      • It is also known as content division element. It just a container to hold
      other tags and group them together

      It is a block element

      w3school TutorialsPoint

    • Span Element

      • It is a generic container used to hold other html elements or group elements
      together.

      It is a inline Element

      w3school TutorialsPoint

  3. Superscript vs Subscript Tags

    • Superscript Tag

      It is a tag which helps us to write formula and powers; where it comes at top
      and are small in size.

      For example: (a+b)2

    • Subscript Tag

      It is a tag which helps us to write formula; where it comes at bottom and are
      small in size.

      For example: H2O

  4. Semantic vs Non-Semantic Markup

    • Semantic Makeup

      It is the makeup that relates to the meaning of the content. It is used to make webpages
      meaningful, and the layout structured. It makes the wepages SEO(Search Engine Optimization)
      friendly. It helps to make website more readable and improve user experience.

      Common Example of Semantic Tags are:
      • Header
      • main
      • Footer
      • Nav

    • Non-Semantic Makeup

      It is the makeup that doesn't relate to the meaning of the content.They can be used with
      different attributes to mark up semantics common to a group.

      Common Exmaple of Non-Semantic Tags are:
      • Div
      • Span
